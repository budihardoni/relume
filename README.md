# Relume OS

Repository awal untuk Relume Component Intelligence: component registry, examples, references, assets, dan export workflow.

## Critical structure

- `index.html` — main entry
- `examples.html` — example asset curation
- `ref.html` — reference card curation
- `set.html` — design setup workspace
- `styles/` — CSS files
- `src/` — browser-side app modules
- `data/` — JSON registries
- `assets/` — Relume previews, examples, and references
- `exports/` — generated handoff outputs
